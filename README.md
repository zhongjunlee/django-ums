#### 介绍
这是一个基于 Django + MySQL + Bootstrap 开发的用户管理系统，帮助你快速上手Python Web开发。

#### 软件架构
前端：Jquery、Bootstrap
后端：Python3.11、Django5.0
数据库：MySQL8.0+
