# django-ums

#### 介绍
基于Django Python Web框架 + MySQL + Bootstrap 开发的用户管理系统, 目前包括：部门管理、用户管理、认证-注册登录、文件上传等功能。

#### 软件架构
前端：Jquery、Bootstrap
后端：Python3.11、Django5.0
数据库：MySQL8.0+
